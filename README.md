# Excel 函数知识库

> 120 个 Excel/表格函数交互演示，从入门到精通

## 📊 项目说明

本项目包含 120 个常用表格函数的交互式演示，每个函数都有：
- ✅ 分步动画演示
- ✅ 实时公式显示
- ✅ 速度控制
- ✅ 手机/微信适配

## 📚 函数分类

- **P0 基础函数** (15 个) - SUM, AVERAGE, IF, VLOOKUP 等
- **P1 常用函数** (25 个) - FILTER, SORT, UNIQUE, TEXT 等
- **P2 高级函数** (20 个) - LAMBDA, LET, MAP, REDUCE 等
- **P3+ 其他函数** (60 个) - 财务、统计、工程等
